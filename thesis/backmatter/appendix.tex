    \section*{Task 1 - Sales Commissions}
    \textbf{Questions}
\begin{enumerate}
    \item What are the User inputs to the cell K3?
    \item Explain in your own words how the final commission is calculated
    \item Zora receives a bonus of 39\$, under which condition would Zora not receive a bonus?
    \item What changes if we change Bob's Sales Target to 15000\$?
    \item There is an error in Levants Base Commission \% correct it
\end{enumerate}

\section*{NASA-TLX Questionnaire}
The participants had to answer all of these questions on a range between 0 and 100.
\subsection*{Question 1}
How mentally demanding was the task?
\subsection*{Question 2}
How hurried or rushed was the pace of the task?
\subsection*{Question 3}
How successful were you in accomplishing what you were asked to do?
\subsection*{Question 4}
How hard did you have to work to accomplish your level of performance?
\subsection*{Question 5}
How insecure, discourage, irritated, stressed and annoyed were you?

\section*{Open-Ended Questions Answers}
\subsection*{Question 1}
What did you like the most about the tool?


\textbf{Participant 1}: "explanations about what a formula does, color coded-ness, it helped me understand the hidden meaning of the numbers on the sheet"


\textbf{Participant 2}: "all relevant information is clearly given in the tool
links to the origin of the data 
clear visual guidence for using"


\textbf{Participant 3}: "The drag and drop system is very appealing and I loved that you can always see in which column you are when you slide over it with your mouse.

It is extremly helpful for users with little to no experience to understand whats going on, because usually in Excel u can only look in the formula bar, which is sometimes confusing."


\textbf{Participant 4}: "great way to visualize, especially when you look the first time into a new spreedsheet. also to navigate and understan quickly formulas"


\textbf{Participant 5}: "I found it very helpful to easily make out the input variables. Also the flow chart made it very easy to understand the logic behind each cell. Especially the navigation by double clicking was very intuitive"

\subsection*{Question 2}
What was frustrating?

\textbf{Participant 1}: "i wish there was a ,clear option to stop displaying everything you were looking at on the right and start over"


\textbf{Participant 2}: "expansion poins are not visible enoug"


\textbf{Participant 3}: "The argument of functions are a bit frustrating, because i couldnt understand what happens in there sometimes and it was a bit like a blackbox for users with very little understanding of spreadsheets and functions."


\textbf{Participant 4}: "in a long tree, sometimes difficult to understand interdependacies, but maybe not caused by the tool, more by complex formulas. sometimes loosed track"


\textbf{Participant 5}: "I found it a bit irritating to open all the variables when searching for the input variables. For example when searching the inputs for the Reccomendation in the Pricing\_Analysis pressing all the + Symbols takes time as the function is fairly complex."

\subsection*{Question 3}
What would you improve?

\textbf{Participant 1}: "see Answer to What was frustrating"

\textbf{Participant 2}: "adaptabl window size, highligte expansion points when unclicked and grey when clicked, shortcut for switching windows"


\textbf{Participant 3}: "I would extend the question mark box with a little more description of what the box is doing."


\textbf{Participant 4}: "maybe make logicgates more intuitiv. sometimes make interdependacies more clear, instead of lable like aand b use more meaningfull name, maybe possible to set a name for them by myself would be nice"


\textbf{Participant 5}: "When using the lookup tables it would be helpful to be able to click on the result and see where in the table it was found."
