\section{Architectural overview}
The system can be divided into four layers.
\begin{samepage}
    \begin{enumerate}
    \item The presentation layer
    \item The visualization layer
    \item The parsing layer
    \item The data layer
\end{enumerate}
\end{samepage} 
The \textbf{presentation layer} is responsible for rendering the split view and dispatching user interactions. 
The second layer, the \textbf{visualization layer}, bridges the underlying formula strucutre and the flow graph by transforming the AST produced by the parsing layer into an interactive graph representation. The \textbf{parsing layer} is responsible for two conversions: parsing formula strings into an AST, and serializing an AST back into a formula string. The foundation of the system is the \textbf{data layer} which hosts the spreadsheet calculation engine and serves as the single source of truth for all spreadsheet data.
